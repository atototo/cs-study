# 화면 설계    
### UI (User Interface)란?
- 사용자와 시스템 사이에서의 의사소통을 할 수 있도록 만들어진 매개체이다.
- 사용자가 시스템을 쉽게 사용하도록 돕는 장치나 소프트웨어를 말한다.

### UI의 종류 
| 종류 | Full Name | 설명 | 특징 |
|------|-----------|------|------|
| **CLI** | Command Line Interface | 텍스트 명령어를 입력하여 조작 | 전문가 위주, 효율적, 키보드 중심 |
| **GUI** | Graphical User Interface | 아이콘, 메뉴 등 그래픽 요소 이용 | 마우스 중심, 직관적, 일반 사용자용 |
| **NUI** | Natural User Interface | 사용자의 신체 부위, 움직임 이용 | 터치, 음성, 동작 등 자연스러운 조작 |
| **VUI** | Voice User Interface | 사용자의 음성으로 명령 수행 | 비접촉 소통, 핸즈프리 (Siri, Alexa 등) |
| **OUI** | Organic User Interface | 모든 사물이 인터페이스가 됨 | 형태 변화, 유연한 디스플레이 등 별도의 장치 없이 모든 사물이나 물질들이 자연 형태 그대로 인터페이스 장치가 되는 인터페이스 |
| **CUI** | Cell User Interface | 표 형식의 데이터 입력 및 관리를 위해 사용하는 셀 기반의 사용자 인터페이스 | 엑셀 |
| **MUI** | Mobile User Interface | 모바일 기기에서의 사용자 인터페이스 | 터치, 음성, 동작 등 자연스러운 조작 |

### UI의 설계 원칙
### 1) 직관성 
> 누구나 쉽게 이해하고 사용할 수 있어야 한다.
### 2) 유효성
> 사용자의 목적을 정확하게 달성할 수 있도록 해야 한다.
### 3) 학습성
> 누구나 쉽게 배우고 익힐 수 있어야 한다.
### 4) 유연성
> 사용자의 요구사항을 최대한 수용, 오류를 최소화 할 수 있어야 한다.
