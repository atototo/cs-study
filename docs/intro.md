## 🎯 1단계 상세: 컴퓨터 시스템 기초

```
[CPU 기초] → [메모리 구조] → [프로그램 vs 프로세스] → [프로세스 vs 쓰레드]
    ✅           🔄                 ⬜                      ⬜
```

### 학습 순서

1. ✅ **CPU 기초** - 코어, 클럭, 캐시
2. 🔄 **메모리 4영역** - Stack, Heap, Data, Code
3. ⬜ **프로그램 vs 프로세스** - 실행 전 vs 실행 후
4. ⬜ **프로세스 vs 쓰레드** - 면접 단골 질문!
5. ⬜ **컨텍스트 스위칭** - 왜 비용이 드는가
6. ⬜ **동기화** - Mutex, Semaphore
7. ⬜ **데드락** - 발생 조건과 해결
